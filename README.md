# QuickBlur Releases

A macOS native app to add blur effects to sensitive data on your images.

## Download

**[Download Latest Release](https://github.com/rockiestar-com/quick-blur-releases/releases/latest)**

Or download directly:
- [QuickBlur-latest.dmg](https://github.com/rockiestar-com/quick-blur-releases/releases/latest/download/QuickBlur-latest.dmg)

## System Requirements

- macOS 15.0 (Sequoia) or later

## Auto-Updates

QuickBlur includes automatic updates via Sparkle. The app will check for updates daily and notify you when a new version is available.

**Appcast URL:** `https://github.com/rockiestar-com/quick-blur-releases/releases/download/appcast/appcast.xml`

## Installation

1. Download the DMG file
2. Open the DMG and drag QuickBlur to your Applications folder
3. Launch QuickBlur from Applications

## Features

- Drag & drop image loading
- Brush-based selective blur tool
- Auto text detection and blur (using Vision framework)
- Zoom and pan controls
- Undo/redo support
- Export to PNG/JPEG/TIFF

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| ⌘O | Open image |
| ⌘E | Export image |
| ⌘Z | Undo |
| ⌘⇧Z | Redo |
| ⌘U | Check for updates |
